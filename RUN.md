###  Создаем sim link на ./cli/run в /usr/local/bin
```shell
sudo ./cli/run in-link
```

### Запуск backend
```shell
mg dc up -d
```

### Применяем миграции
```shell
mg al-up
```



### Стоп проекта
```shell
mg dc down
```

### swagger
https://localhost:42701/docs



### Запуск frontend
```shell
mg cl-run
```
